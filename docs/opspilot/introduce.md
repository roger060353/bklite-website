# 产品介绍

## 产品简介
OpsPilot 是 BK-Lite 打造的一站式企业级智能应用构建平台。它致力于为企业提供从底层模型接入、私域知识管理、智能体编排，到多渠道对话交付的全链路闭环能力。通过高度可配置的工作台与可视化的流程式编排，企业可以快速构建出符合自身业务需求的专属 AI 助手。

## 核心优势
*   **「零代码」极速构建**：通过直观的可视化拖拽引擎与 ChatFlow 画板，无需深入代码细节即可完成智能体逻辑编排与发布。
*   **私域知识深度融合**：内置灵活强大的知识库能力，支持各类文档沉淀、问答对抽取与知识图谱构建，让 AI 清晰理解企业专属业务上下文。
*   **全场景多渠道交付**：一次编排，多端可用。内置对企业微信、钉钉、Web、移动端 APP 等主流渠道的接入支持，打破沟通界面壁垒。
*   **大模型生态兼容**：统一管理底层 LLM、Embed、Rerank、OCR 等模型，不受单一厂商绑定，根据业务需求灵活调度不同参数侧重的大模型。
*   **企业级安全与运维**：具备缜密的权限分组管控与敏感配置加密机制，并提供全景的会话审计与 Token 用量分析，保障接入过程安全可控。

## 应用场景
*   **智能研发助手**：对接企业内部 Git 仓库或研发平台，利用知识图谱自动分析代码结构或异常日志，帮助研发人员快速定界排障。
*   **智能 IT 客服**：提取企业 IT 服务规范与常见 Q&A 形成知识库，在企业微信中提供 7x24 小时的自动化应答服务，减轻 IT 支持人员重复性工作。
*   **数据查询与流转代理**：借助 OpsPilot 工具集接入企业内部 API，配合 ChatFlow 决策能力，以对话形式帮助业务人员查询指标或流转内部工单。
