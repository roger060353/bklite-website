# 快速入门

本文将指导您从首次登录起，快速掌握如何自定义您的主页大纲，接收平台的重要信息，并管理自己底层账号的偏好设置。

## 前置条件
*   已获取到 BK-Lite 平台的正式基础访问账号及原始密码。
*   系统管理员已为您完成基础身份注册并分配了特定应用使用（如监控、OpsPilot等）的查看角色。

## 分步操作指引

### 1. 首次登录与卡片订阅
当您拥有合法凭证，第一次进入门户时的准备动作。
1.  在登录页面通过您的系统凭据访问后，由于属于全新账户，可能会看到初始化的首次加载提示窗。按系统默认指引点击完成基础装配。
2.  进入主页后，您可能看到按系统下发规则呈现的默认几张“应用卡片”（例如“基础监控”或“CMDB”）。
3.  点击页面上的「自定义设置」等类似功能按键。
4.  在应用排序选单中，您可以将目前用不到的偏门应用拖拽隐藏，或者将自己所处岗位需要最高频访问的主导系统（如“运维工作台”）卡片拖拽排在最前方第一序列。最后点击「保存」。

### 2. 账号与个人偏好管理
确保通知能够投递给正确的邮箱并保持国际化时区对齐。
1.  找到右上角的个人头像并点击，在下拉悬浮菜单中选择进入「个人信息」管理页。
2.  如您跨国区协同开发，可以在此修改页面展示「语言」及系统默认挂载「时区」，保证数据与日志时间的显示不产生错位落差。
3.  如有必要，您可以修改与个人账号强绑定的邮箱，在此过程中您需要接受旧邮箱验证码或身份凭据复核的安全校验。

### 3. 处理历史密码合规
基于公司审计要求或者出于定期保养对弱口令进行修补加固。
1.  同样在个人信息的安全管理栏目内，点击修改密码。
2.  根据前端抛出的动态“安全策略要求”（如可能要求：最少12位、包含特殊字符与字母大小写等强约束条件）进行修改。
3.  保存并建议重新尝试登录。

## 结果验证与闭环

完成装配与系统磨合后，可以在日常工作中进行验证与收口使用：
1.  刷新当前系统控制台首页即可观察：您所配置的核心看板应用是否已经排列在了您的视觉第一位，无关的卡片是否已经被完全过滤出可视范围。
2.  请随机点击任一应用并进入，从左上角的图标即可快速切回这扇“主控制大门”。
3.  当你离开主站或一段时间重新登入内页，注意到控制台顶部如果出现红色或数字角标提醒消息时。点开头像旁边的“消息小铃铛”，在弹出的「通知中心」中查看您需要跟进的具体系统知会事项。如果您确认该批事件无潜在风险，可以直接点击“标记全部已读”完成业务审计流的跟进闭环。
