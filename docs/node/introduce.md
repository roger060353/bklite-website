# 产品介绍

## 一、产品简介
节点管理是基于 BlueKing Lite 构建的的**节点与探针管理中枢**，致力于云区域下节点的集中管控。通过为各类节点统一部署、管理探针与组件，为监控数据采集、配置信息同步等业务场景提供基础支撑。
凭借轻量化架构与标准化配置体系，节点管理模块实现了多云区域、异构节点环境下的统一管理，大幅简化云资源与探针的全生命周期运维流程。

## 二、产品核心优势
1. **云节点集中管控，打破资源壁垒**
   统一管理多区域网络下的节点资源，可视化展示各区域节点分布与状态，消除分散管理的壁垒，精准掌握全局节点资产。 

2. **自动化运维提效，降低手动成本**
   依托Linux/Windows控制器，实现节点上探针、组件的部署、配置、维护全流程自动化。结合基于主次隔离的标准化配置下发体系，核心操作无需繁琐人工干预，显著提升规模化运维效率。
   
3. **探针与组件生态完整，适配多场景**
   内置开放式的组件库管理体系，提供如Telegraf、Beats等丰富的探针与Agent版本包，兼容Linux、Windows等异构环境，全面满足监控、日志、CMDB等多样化业务下发需求。

## 三、产品应用场景
- **多云区域资源统一管理**
  集中管控不同云区域节点资源，精准掌握各区域节点分布、状态与关联组件，实现云资源全局可视与统一运维。

- **探针全生命周期管理**
  加速节点各类探针的部署、配置与维护，为监控采集、日志收集、指标上报等业务场景奠定基础。

- **跨节点运维自动化**
  借助控制器实现多节点探针、组件的批量部署、升级与管控，满足大规模节点集群运维需求，提升规模化运维效率。

- **异构环境适配支撑**
  全面兼容Linux、Windows等多系统节点，适配多架构资源管理与探针部署，灵活支撑异构环境业务落地。
