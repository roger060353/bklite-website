# 控制台 产品介绍

## 产品简介
控制台（Console）是用户每天登录 BK-Lite 平台后看到的第一个枢纽型界面。它承担着全平台统一入口、全局消息收发中心以及个人设置大本营的角色。控制台不仅仅是一个菜单的集合，它被设计为一个高度可定制化且跨端的“个人专属工作台”，致力于为不同岗位的 IT 或业务人员提供最短路径的信息到达体验。

## 核心优势
*   **统一鉴权的中央门户**：作为跨越底层各子系统的入口桥梁，它通过无缝对接底层的权限引擎，做到在登录后仅对用户呈现其拥有访问权的应用，规避业务视角的干扰和安全越权。
*   **私有化的门户卡片**：不同部门员工关注的核心系统截然不同（如研发关注大模型与发布，运维关注监控与告警）。控制台提供了灵活的可视化拖拽机制，每位员工都能根据自己的业务高频流向，自主订阅、屏蔽及排序首页模块卡片。
*   **全景收口的跨模块消息总线**：汇聚由告警模块、系统权限变更甚至模型训练完成投递过来的所有业务通告。具备一站式的批量管理追踪能力，使得用户彻底告别被淹没在零散内部邮件中的困扰。
*   **端云一致的用户管理**：对个人资料与安全选项（邮箱换绑、语言偏好、密码轮转等）的统一管理承接点。并支持了移动端 APP 及 Web 浏览器的双端接口同源同步，确保员工信息的全天候准确和一键全系统生效。

## 应用场景
*   **敏捷业务路由下发**：在新员工入职平台或组织下发新系统时，首登的新手能够直接通过控制台完成默认属性编排。并在日常工作中，使用此门户实现 CMDB 到 监控 等各不同子应用的高速跳板切换。
*   **全局待办梳理与溯源**：处理如审批告警或系统权限审计时，用户从控制台的消息通知铃铛出发，了解自己需要复核的重要事项详情并完成批量“已批阅”闭环操作。
